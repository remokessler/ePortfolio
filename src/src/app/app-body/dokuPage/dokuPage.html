Zur Umsetzung der Webseite gab es nicht viele Überraschungen. Die einzige die ich angetroffen habe war ein Materialize CSS und Angular 2 Inkompatibilität, so können z.B. keine Modale standardmässig aufgerufen werden. So musste ich eine eigene Instanz von $ global definieren und die Modalklasse von hand schreiben.
